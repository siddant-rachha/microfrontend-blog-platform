# **Micro-Frontend Blog Platform --- Unified Repository**

A complete micro-frontend powered **blog platform ecosystem** consisting
of:

-   A **Next.js surface application**
-   A **shared UI components library** published to npm
-   A **web components bundle** consumable across frameworks
-   A **React Native mobile app**
-   A deployed **live production blog**

This repository consolidates all related links, documentation, and
resources for the entire system.
